# 芳华流逝

芳华流逝，帮助您更好地感知时间的流逝。

## 功能特点

- 显示已活时间（生之钟）
- 显示剩余时间（死之钟）
- 实时更新的时间显示
- 计算还能做的事情次数
- 可自定义的生日和预期寿命
- 美观的界面设计

## 安装方法

1. 打开 Obsidian 设置
2. 进入第三方插件
3. 关闭安全模式
4. 点击浏览社区插件
5. 搜索 "芳华流逝"
6. 点击安装
7. 启用插件

## 使用方法

1. 点击左侧工具栏的时钟图标，或使用命令面板中的"切换芳华流逝"命令
2. 在插件设置中配置您的生日和预期寿命
3. 点击时钟界面可以切换显示模式（生之钟/死之钟）

## 设置选项

- **生日**：设置您的出生日期（格式：YYYY-MM-DD）
- **预期寿命**：设置预期寿命（年）

## 默认活动

插件默认设置了以下活动：
- 阅读（每周一次）
- 旅行（每年两次）
- 写作（每月一次）
